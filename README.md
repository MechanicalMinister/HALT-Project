# HALT-Project

File sharing and storing location for the HALT project — a Master's project at Aalborg University.

## Repository Structure

```
HALT-Project/
├── code/           # Source code and scripts
├── data/           # Datasets and raw data files
├── docs/           # Documentation and meeting notes
├── presentations/  # Slides and presentation materials
└── reports/        # Project reports and papers
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/MechanicalMinister/HALT-Project.git
   ```
2. Add your files to the appropriate folder.
3. Commit and push your changes:
   ```bash
   git add .
   git commit -m "Describe your change"
   git push
   ```

## Contributing

All project members can contribute by pushing files to the relevant directories. Please use descriptive commit messages and keep files organized within the appropriate folder.
